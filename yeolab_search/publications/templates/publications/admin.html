Bulk Update

Fetches new publications from PubMed, links GEO/SRA datasets, or imports ENCODE experiments. Existing records are not overwritten — only new papers and datasets are added.

ENCODE fetches experiments for grants U41HG009889 and U54HG007005 from encodeproject.org. Methods extraction scans abstracts, keywords, GEO, and SRA metadata for computational tools — no external API access needed. Analysis extraction parses ordered processing steps from GEO Data Processing sections and PMC full-text Methods — requires NCBI access.

Upload a JSON file exported from ENCODE search (for example: /search/?type=Experiment&format=json&limit=all&award.name=U41HG009889). The importer will parse experiments, link datasets/publications, and sync processing-step analysis.
